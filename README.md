# 🏗️ Перевернутая башня — задача о нумерации комнат

В новом кампусе университета запланировали построить общежитие — удивительную башню, расширяющуюся к верху, чтобы всем комнат хватило, а если понадобятся ещё комнаты, то новый этаж можно будет пристроить.

## 🏢 Устройство башни

Башня устроена следующим образом:
- **1-й этаж** — 1 комната
- **2-й и 3-й этажи** — по 2 комнаты на каждом
- **4, 5, 6 этажи** — по 3 комнаты на каждом
- **7, 8, 9, 10 этажи** — по 4 комнаты на каждом
- и так далее...

На каждом этаже комнаты нумеруются слева направо.

Башню решили оборудовать лифтом — и вот задача: студентам нужно научиться по номеру комнаты определять, на каком этаже она находится и какая она по счету слева на этом этаже.

## 📥 Входные данные

Одно целое число — номер комнаты `N` (1 ≤ N < 2<sup>64</sup>)

## 📤 Выходные данные

Два целых числа, разделенных пробелом:
- Номер этажа
- Порядковый номер комнаты слева на этом этаже

## 📋 Пример

**Вход:**

```
13
```

**Выход:**

```
6 2
```

**Пояснение:**

```
Этаж 6: 12 13 14
Этаж 5: 9 10 11
Этаж 4: 6 7 8
Этаж 3: 4 5
Этаж 2: 2 3
Этаж 1: 1
```


Комната №13 находится на 6-м этаже и является второй слева.

## 🛠️ Алгоритм решения

Задача решается в три этапа:

1. **Поиск группы** — определяем, в какую группу этажей попадает комната (группа `k` — это `k` этажей по `k` комнат)
2. **Позиция внутри группы** — вычисляем этаж внутри группы и номер комнаты на этаже
3. **Глобальный этаж** — суммируем этажи предыдущих групп и этаж внутри текущей

### Сложность
- Временная: O(∛N) — количество групп растёт как кубический корень от номера комнаты
- Память: O(1) — только несколько переменных
